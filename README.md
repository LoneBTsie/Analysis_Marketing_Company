# Product Funnel Analysis for Marketing Company
This project uses data from Microsoft Build 2026 databases to perform a funnel analysis across key ecommerce steps:

HomePage → ProductPage → Add to Cart → Checkout

With 4,000 sessions, this dataset offers a rich opportunity to identify conversion drop-offs, assess user behavior by segment, and suggest improvements based on actual funnel results.

*Skills: SQL, Python, PowerBI*

Check out the PDF file and the scripts for detailed analysis.
